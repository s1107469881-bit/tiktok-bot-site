My Content Bot

An AI-powered content automation tool that helps creators schedule and publish TikTok videos using AI-generated scripts and voiceovers.

This tool is built on the n8n workflow automation platform and uses the TikTok Content Posting API to streamline content creation for individual creators.

About

My Content Bot automates the TikTok content creation workflow by:

  • Generating video scripts using AI language models
  • Creating natural-sounding voiceovers with AI text-to-speech
  • Synthesizing complete videos with FFmpeg
  • Scheduling and publishing content at optimal times
